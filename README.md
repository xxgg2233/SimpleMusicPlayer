# SimpleMusicPlayer
一个简单的音乐播放器
实现一个功能基本完整的音乐播放器程序。

实现了基本的播放暂停功能

上下曲功能，并且会标注正在播放的歌曲

可以根据播放模式（随机/顺序/列表/单曲）自动播放下一曲
